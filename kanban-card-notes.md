# Kanban Card Visual Reference Notes

## Image 1 — Card Layout with Overdue Task Highlight
- Cards have white background, rounded corners, subtle border
- Top: status badge (ex: "Nova" com quadrado azul) + ícone "i" (info circle) no canto superior direito
- Title: bold, ex: "A10X – Alessandra"
- Below title: calendar icon + date "28/02/2026" | megaphone icon + "Início de Checkout – A..."
- **OVERDUE TASK ROW**: fundo ROSA/PINK (bg-pink-200 ou similar), rounded, full width
  - WhatsApp icon + "Primeiro contato" + date/time "28/02/2026 22:07"
  - This is the key visual: the task line itself has a pink/rose background

## Image 2 — Info Popup "Sobre a Negociação"
- Title: "Sobre a Negociação" with X close button
- Separator line
- Section header: "DADOS GERAIS" in teal/dark color, uppercase
- Fields (label in gray, value in bold black):
  - Fonte: "Desconhecido"
  - Campanha: "Início de Checkout – Acelera10X"
  - Interações: "0 interações"
  - Última anotação: "Nenhuma anotação foi feita para essa negociação"
  - Data de criação: "28/02/2026 às 21:07"
  - Último contato: "Não preenchido"
  - Previsão de fechamento: "Não preenchido"
  - Identificador: (cut off)
- Bottom: teal button "Abrir Negociação"
